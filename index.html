Registro de Proveedores
(CRUD)

El proyecto de Registro de Proveedores (CRUD) fue creado para demostrar las habilidades al momento de utilizar el lenguaje de Javascript en conjunto con Bootstrap, para generar de manera dinámica un sistema de registro de usuarios. Los usuarios como data predeterminada son consumidos desde el endpoint “users” del sitio {JSON} Placeholder mediante una función async/await con fetch. El código fue segmentado en módulos para la creación, modificación, eliminación y validación de registros.

Habilidades aplicadas

HTML CSS Bootstrap JavaScript
